# AppLoginCompose
 Creating a mobile app for login (sign in) and register (sign up) account using Jetpack Compose with local data storage.

 ## Features

- Kotlin
- ViewModel
- Navigation
- Coroutines
- Dagger Hilt
- Room Database
- Jetpack Compose
- MVVM Architecture
